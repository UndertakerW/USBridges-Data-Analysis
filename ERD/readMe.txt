The ERD is generated using "MySQL Workbench 8.0 CE". It can be generated by clicking ''Database"->"Reverse Engine".

Notice that, the ERD you generated may have some bugs: the primary-foreign key may just disappear. To fix that, you can refer to this link: https://github.com/horeca2/MySQL-Workbench-Bug-92141-patch
